## 接线图

### ESP32-S3 引脚分配

| 功能 | GPIO | 说明 |
|------|------|------|
| WS2812B Data | GPIO8 | 眼睛LED数据线 |
| 舵机1水平 | GPIO6 | 水平转动 |
| 舵机2垂直 | GPIO7 | 垂直转动 |
| INMP441 SCK | GPIO15 | I2S时钟 |
| INMP441 WS | GPIO14 | I2S字选择 |
| INMP441 SD | GPIO13 | I2S数据 |
| MAX98357 BCLK | GPIO41 | I2S位时钟 |
| MAX98357 LRC | GPIO42 | I2S左右声道时钟 |
| MAX98357 DIN | GPIO40 | I2S数据输入 |
| HC-SR04 Trig | GPIO9 | 超声波触发 |
| HC-SR04 Echo | GPIO10 | 超声波回响 |
| 摄像头 | 特定引脚 | 见ESP32-CAM引脚图 |

- **3.7V锂电池**: 供给ESP32直接使用

## 购买建议

### 推荐供应商

1. **立创商城** (lcsc.com) - 电子元件
2. **淘宝/天猫** - 综合采购
3. **京东** - 快速配送
4. **拼多多** - 低成本选项

### 购买优先级

#### 第一批（核心功能）
- ESP32-S3开发板
- WS2812B LED矩阵
- SG90舵机 x2
- I2S麦克风
- USB转UART

#### 第二批（音频输出）
- MAX98357A功放
- 扬声器

#### 第三批（视觉增强）
- OV2640摄像头
- HC-SR04超声波

#### 第四批（完善）
- 电池和充电模块
- 3D打印外壳
- 其他传感器

---

## 成本优化建议

1. **使用现有设备**: 主机端可用现有电脑/笔记本
2. **批量采购**: 多件同类型可降低单价
3. **替代方案**: 
   - 舵机可用MG995替代（更大力矩，更贵）
   - LED可用LCD屏幕替代（更贵但更丰富）
   - 摄像头可用USB摄像头+USB扩展
